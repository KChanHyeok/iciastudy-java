키보드로 입력받는 방법

객체, 인스턴스, 클래스(class)
  객체(Object) : 개념. "자동차"
  인스턴스(instance) : 객체를 실체화 한 것.(생성된 상태) "소나타"
  클래스(class) : 개게를 정의해 놓은 것.(인스턴스를 만드는 설계도)

psvm : public static void main() 생성 단축키
sout : System.out.println() 생성 단축키

변수 : 데이터를 저장하는 공간.
  변수의 선언>
    자료형 변수명;
  변수의 초기화>
    자료형 변수명 = 초기값;
    자료형 변수명;
    변수명 = 초기값;

자료형(Data Type) : 기본형 8개와 참조형으로 구성
    1) 숫자형
        정수형 - byte(1), short(2), int(4), long(8)
        실수형 - float(4), double(8)
    2) 문자형 - char(2)
    3) 불(boolean)형(논리형) - boolean(1)

    참조형 String (문자열형) - 크기 제한 없음.

 컴퓨터에서의 단위
 1) bit - 0/1 한자리.
 2) byte - 8 bit -> 데이터의 최소단위

 부동소수점 - 0.1 = 0.1000 -> float

 참조형이란?
    메모리 상에서 변수의 위치와 데이터의 위치가 다른 상태.
    통상적으로 변수는 호출스택 영역에 생성되고, 데이터는 힙 영역에 생성된다.
    이 때 변수에는 데이터의 위치정보(주소)가 저장되어 이 위치정보로 실제 데이터를 참조한다고 해서
    참조형이라고 한다.
